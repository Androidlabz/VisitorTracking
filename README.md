# VisitorTracking
Android app to track visitors count and show live comments

Add the JitPack repository to your build file

allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
	
	dependencies {
	        implementation 'com.github.androidlabz:VisitorTracking:1.0.3'
	}
